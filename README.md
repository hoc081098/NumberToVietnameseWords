# NumberToVietnameseWords
Example:
```kotlin
println(java.text.DecimalFormat("###,###").format(Long.MAX_VALUE) + " = " + NumberToVietnamese.convert(Long.MAX_VALUE))
// 9,223,372,036,854,775,807 = Chín tỷ tỷ hai trăm hai mươi ba triệu tỷ ba trăm bảy mươi hai nghìn tỷ không trăm ba mươi sáu tỷ tám trăm năm mươi tư triệu bảy trăm bảy mươi lăm nghìn tám trăm lẻ bảy
```
